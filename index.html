Sustainable Autonomy Research Project, Founder

At U.C. Santa Cruz, waste management has become a large problem; my team and I are fixing that problem. We successfully applied for a research grant have created a waste bin which sorts waste autonomously to reduce contamination in recycling, increase compost yields, and reduce the number of items sent to the landfill. Using the AWS rekognition, we have successfully been able to classify objects as trash, recycling or compost. We used a Raspberry PI for the computer to control the robotics which physically sorts the goods.

Fairness in AI, ML model

My group and I trained an ML model to find data discrepancies in the popular Adult dataset hosted on the UCI ML database. We found and trained the optimal model and then tested it against real world data. Using accuracy and error as metrics, we learned that this dataset, which is supposed to represent the world population, consists of mostly white men which caused it to predict that minorities will make less than <50k a year over 80% of the time.

Profanity Checker API

Designed and wrote a RESTful API, in Python, which utilized an SQL database to search for inappropriate words and phrases. This project used a python package in addition to the SQL database to search for profane words in a JSON file. The SQL database is able to be modified using HTTP requests and the profanity check is executed by a POST request.

Voice Activated Waste Sorter

During Cruzhacks, my team and I created a voice activated waste sorter. We used a button connected to Arduino to trigger a Python app to record audio. The audio is parsed using Google Cloudâ€™s speech API, and then compared to our database of items hosted on Microsoft Azure. When a decision is made, the python app sends a serial signal to the Arduino, which turns a servo motor, opening the correct receptacle and closing the wrong one. Using the Notivize API, a message is sent when the can is full.

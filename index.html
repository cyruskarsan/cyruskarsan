Profanity Checker API

Designed and wrote a RESTful API, in Python, which utilized an SQL database to search for inappropriate words and phrases. This project used a python package in addition to the SQL database to search for profane words in a JSON file. The SQL database is able to be modified using HTTP requests and the profanity check is executed by a POST request.

Voice Activated Waste Sorter

During Cruzhacks, my team and I created a voice activated waste sorter. We used a button connected to Arduino to trigger a Python app to record audio. The audio is parsed using Google Cloudâ€™s speech API, and then compared to our database of items hosted on Microsoft Azure. When a decision is made, the python app sends a serial signal to the Arduino, which turns a servo motor, opening the correct receptacle and closing the wrong one. Using the Notivize API, a message is sent when the can is full.

Sustainable Autonomy Research Project, Founder

At U.C. Santa Cruz, waste management has become a large problem; my team and I are fixing that problem. We successfully applied for a research grant and are creating a waste bin which sorts waste autonomously to reduce contamination in recycling, increase compost yields, and reduce the number of items sent to the landfill.

Ethereum Mining Rig

Before the Bitcoin boom, I learned about blockchain technologies and saw its potential to disrupt industries. In an attempt to support the network and mine coins, I built a ETHash miner using off the shelf hardware. Booted linux onto the miner and used SSH commands to run the scripts.
